# Описание проекта

Проект Dactylus это пополняемый словарь русского жестового языка.
У него есть страницы:
* Гравная
* Словарь с 
* Страница добавления категории
* Страница добавления слова
* Поиск
* Страница слова
* Страница жеста
* Мой словарь
* Модерация
* Переводчик
* Авторизация
* Регистрация


## Главная
Содержит заголовок, под ним: слева текстовое описание проекта, справа то же описание но на жестовом, в виде плеера для mpp4. Видео квадратное в кружке.
Под описанием возможности платформы в виде плиток, среди которых есть не активные.
Внизу новости проекта.


## Словарь с категориями + поисковая строка
Карточка категории:
* Название
* Кол-во слов внутри
* Внизу несколько примеров слов/подкатегорий

Карточка слова:
* Название
* Для жеста - гифка обрезанная под круг
* Описания смыслов маркерами с точками

### Если базовая категория:
Сверху строка поиска.
Под строкой поиска миниатюрные плитки с категориями
Под ними полноценный выбор категорий с пагинацией.
### Если выбрана категория
Вверху навигация через родительские категории, ниже строка поиска.
Ниже слева миниатюрная круглая иконка категории, правее заголовок с названием категории, под ними плитки со словами или подкатегориями с пагинацией.

Добавить кнопку создания категории или добавления слова (сделай обе).


## Страница добавления категории
Форма:
1. Название категории
2. Родительская категория
3. Выбор, после какой категории ее поставить, порядок (динамически через ajax)
4. Картинка категории
* добавить скрытые выводы ошибок


## Страница добавления слова/жеста
Форма:
1. Название слова/жеста
2. Категория
3. Выбор привязанного смысла (поиск через ajax)
4. Выбор Морфологически-базового слова (наследуется ОТ) (поиск через ajax)
4. Выбор стилистически-базового слова (наследуется ОТ) (поиск через ajax)
* добавить скрытые выводы ошибок

## Страница слова/жеста
Сверху слева заголовок слова, справа кнопка "В мой словарь". Под заголовком слова галочка "Проверено" или "Не проверено" (зеленый/серый).
Ниже:
Слева: видеозапись жеста слова, квадратный блок со скругленными углами, красивый плеер html5. Под видеозаписью "приклеенное" переключение между жестами (если несколько), под переключением слайдер для скорости воспроизведения 0.25x - 1.5x.
Справа: карточки слова/жеста
1. Значения - список (для жестов это список из миниатюр записей жестов в gif с объяснениями)
* {СМЫСЛ} - {Описание текстом или жестовой миниатюрой}
2. Синонимы, сгруппированные по смыслам (текстом или миниатюрами)
3. Морфологические связи - слева входящие (происходит ОТ), справа исходящие  (текстом или миниатюрами)
4. Стилистические связи - слева входящие (происходит ОТ), справа исходящие  (текстом или миниатюрами)
Если жестовые миниатюры по кол-ву не влезают по ширине - сделать слайдер с красивой прокруткой.

Сделай две страницы, для слова и для жеста.
